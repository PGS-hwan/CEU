CustomEnchantmentUpgrader 是一款玩家物品修复与附魔升级的 Bukkit/Spigot 插件，支持 XConomy 和 PlayerPoints 两种经济系统。

## 兼容环境

- Java 8 及以上
- Spigot / Paper 服务器
- 兼容版本：1.8 - 1.12.2
- 软依赖：XConomy、PlayerPoints
